# 简兮阅读器 2.0.1 (Build 103) 更新报告

## 更新重点

本次更新继续统一全局 UI 语言，处理首页冗余信息、卡片圆角、拖拽反馈，并在关于应用页面加入缓存清理能力。

## 具体改动

- 首页
  - 删除标题下方的「文档数量 / 显示数量」统计胶囊。
  - 保留搜索、排序、导入文档和文档列表逻辑不变。

- 触感反馈
  - 毛玻璃分段选择控件在拖拽切换选项时加入选择震动反馈。
  - 底部导航栏横向拖拽切换首页/设置时加入选择震动反馈。

- 卡片圆角
  - 文档卡片、设置一级页面卡片等 `AppCard` 统一调整为更圆润的 `AppRadii.lg` 圆角。
  - 书架文档卡片同步使用更圆润的圆角。

- 关于应用
  - 新增「缓存清理」卡片。
  - 支持清理临时目录缓存、内存图片缓存和已下载的更新 APK。
  - 不删除用户导入或引用的文档。

- 版本同步
  - 应用版本更新为 `2.0.1+103`。
  - 更新检查参数同步为 `local=103`。
  - 测试中的版本展示同步为 `版本 2.0.1 (103)`。
