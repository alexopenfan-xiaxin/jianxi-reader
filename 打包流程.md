# 简兮阅读器 ARM64 APK 打包流程

代码完成后，执行以下两步即可打包 ARM64 APK：

## 1. 获取依赖

```bash
flutter pub get
```

## 2. 构建 ARM64 Release APK

```bash
flutter build apk --target-platform android-arm64 --release
```

## 输出

```
build/app/outputs/flutter-apk/app-release.apk   (~9.9MB)
```

## 环境要求

- Flutter SDK: `C:\Users\admin\Downloads\tools\flutter`
- Android SDK: 已包含 platform-35 + build-tools 35.0.0 + NDK 27.0.12077973
- JDK: JBR 21 (随 Flutter 捆绑)
- 工作目录: 项目根目录

## 注意事项

- `flutter pub get` 首次运行可能因网络问题较慢，会缓存到本地 Pub Cache
- `flutter build` 首次运行需下载 Gradle，后续增量构建较快
- 如果需要分 ABI 构建（更小 APK）：`flutter build apk --target-platform android-arm64 --split-per-abi --release`，输出为 `app-arm64-v8a-release.apk`
